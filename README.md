# Dũng Sĩ Giải Cứu Công Chúa (React + Three.js)

Game mini chạy trên web với cốt truyện dũng sĩ giải cứu công chúa.

## Tính năng
- Dựng không gian 3D bằng Three.js.
- Điều khiển dũng sĩ bằng bàn phím (WASD / mũi tên).
- Thu thập tinh thể ánh sáng và né bầy sói.
- Gặp công chúa, hộ tống đến cổng dịch chuyển để hoàn thành nhiệm vụ.
- Bảng HUD hiển thị kịch bản và trạng thái nhiệm vụ theo thời gian thực.

## Chạy dự án
```bash
npm install
npm run dev
```

## Build production
```bash
npm run build
npm run preview
```
